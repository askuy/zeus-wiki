# 使用说明

## 使用

> 首页展示的内容 是 当前用户 有登陆权限的系统.

![图片](/guide/1.png)


当用户点击想进去的`项目` ,浏览器则会新建一个`Tab`自动导航到对应的`URL`，并且无需再重复登陆动作。

当前测试兼容的浏览器为：

Chorme V60 or later

Firefox V50 or later

IE V10 or later

Safari 10.0 or later
